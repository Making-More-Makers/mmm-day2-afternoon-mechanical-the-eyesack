# 3D Printing Assignment (STL)

## 1. My Model | 我的模型

- **Model Type | 模型类型**: Mario Pipe Pen Holder
- **Model Dimensions | 模型尺寸**: 70mm × 70mm × 85mm
- **Minimum Wall Thickness | 最小壁厚**: 2 mm

## 2. Design Concept | 设计概念

I created a pen holder in the shape of a Mario Pipe, with a section on the top for storing small objects.

## 3. Modeling Steps | 建模步骤

1. Started with a hallow cylinder as base
2. Added another hallow cylidner on top
3. Made a small platform on top
4. Grouped everything and exported as STL

## 4. Printability Checklist | 可打印性检查清单

- [x] Model is a closed solid
- [x] Wall thickness ≥ 1.2 mm
- [x] No overhangs > 45°
- [x] Model sits flat on workplane
- [x] Size ≤ 50 mm
- [x] All shapes grouped
- [x] Exported as STL
